1、数据库为mysql，在mysql中新建一个“filesharing”的数据库，再运行filesharing.sql的脚本进行建表和添加数据的操作
2、在eclipse JavaEE中选择Window->Preferences->General->Workspace，在其中将默认编码格式改为UTF-8
3、之后就可以直接在eclipse中选择File->Open Projects Form File System进行添加项目
4、右键项目Run As->Run On Server选择已有的Tomcat即可运行
5、在浏览器中访问http://127.0.0.1:8080/FileSharing/first.jsp默认账户密码为root和456
